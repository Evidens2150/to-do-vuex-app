# ToDo List App

# Vue 3 + TypeScript + Vuex

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
### Build version preview
```
npm run preview
```

### Deploy to GitHub Pages
```
npm run deploy
```
